# 画像ディレクトリ

このディレクトリには、UserGuide.md および README.md で使用されるスクリーンショットや図を配置します。

## 必要な画像ファイル

以下の画像を追加することで、ユーザーガイドがより視覚的でわかりやすくなります:

1. **top-screen.png** - BondLogのメイン画面(トップ画面)
2. **add-platform.png** - プラットフォーム追加フォーム
3. **add-stream.png** - 配信追加フォーム
4. **add-listener.png** - リスナー追加フォーム
5. **add-participant.png** - 配信詳細画面から参加者を追加
6. **listener-detail.png** - リスナー詳細画面の例
7. **edit-listener.png** - リスナー編集フォーム
8. **add-gift.png** - ギフト追加フォーム
9. **status-management.png** - ステータス管理画面
10. **status-badge.png** - ステータスバッジの表示例
11. **menu.png** - メニュー画面(エクスポート/インポート)

## 画像の要件

- **形式**: PNG形式を推奨
- **サイズ**: 幅800px程度が適切
- **内容**: 実際のアプリケーション画面のスクリーンショット
- **プライバシー**: 個人情報が含まれないようサンプルデータを使用

## スクリーンショットの撮り方

1. アプリケーションをブラウザで開く
2. 各機能画面を表示
3. ブラウザの開発者ツールやスクリーンショットツールで撮影
4. 必要に応じて注釈や矢印を追加
5. このディレクトリに配置
